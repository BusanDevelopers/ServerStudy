## 프로젝트

스터디에서 학생들이 만든 프로젝트들 입니다.

### 음식 알림기

1. 바코드의 정보와 공공 데이터 API를 통해 음식의 정보를 불러와서 등록한다.

2. 등록된 음식의 유통기한이 임박하면 사용자에게 알려주고, 관련 레시피를 불러와 보여준다.

개발자 : 김현우 님
